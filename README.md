# QRCode Tool

QRCode Generator and Reader / 二维码生成和解析工具

## 相关命令（Commands）

初始化 / Initialization

```text
pnpm install
```

启动开发服务器（开发模式）/ Start devServer (Development Mode)

```text
pnpm run serve
```

启动开发服务器（生产模式）/ Start devServer (Production Mode)

```text
pnpm run preview
```

打包 / Build

```text
pnpm run build
```
